# 📦 Order Management System (CLI)
This program is a Python console application designed to manage business sales workflows. 
It allows for the administration of customers, products, and orders through an interactive menu interface.
## 🚀 Key Features
The system is divided into logical modules to ensure data integrity:
1.  **Customer Registration:** Stores unique ID, name, and email address.
2.  **Product Catalog:** Manages products along with their respective unit prices.
3.  **Order Management:** 
    - Validates the existence of both customer and product before creating an order.
    - Automatically calculates the total based on the quantity entered.
    - Prevents duplicate order IDs.
4. **Reports and Queries:**
    - Detailed visualization of all orders.
    - Calculation of total daily revenue.
    - Statistical reporting (orders per customer and total units sold).
## 🛠️ Technical Requirements- **Language:** Python 3.x- **Data Structures:** Uses dictionaries to ensure efficient record lookup by ID.
- **Robustness:** Implements `try-except` blocks to prevent unexpected crashes caused by incorrect data entry.
